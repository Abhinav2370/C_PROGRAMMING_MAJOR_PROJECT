# C_PROGRAMMING_MAJOR_PROJECT
The Railway Ticket Management System is a console-based C program that allows users to book, view, search, and cancel railway tickets.
It uses arrays, structures, and functions to store and manage ticket data in a simple format.

This project is ideal for first-year (beginner) C students, showcasing core concepts like:

Structures

Functions

Conditional statements

Loops

Basic data storage

Menu-driven programming
